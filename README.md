# 🚀 DevTinder: The Tinder for Developers

Ever wished you could just swipe right on the perfect coding partner? **DevTinder** is a platform I built to help developers connect, collab, and grow their network in a fun, intuitive way. No more stiff LinkedIn requests—just profiles, skills, and matches.

## ✨ Why I built this
As a developer, I realized finding people to work on side projects or even just finding a mentor is harder than it should be. I wanted to create something that feels modern and fast, using the same swipe-to-connect mechanics we all know, but specifically for the dev community.

## 🛠️ What's inside?
- **Smooth Swiping**: A Tinder-like interface to discover developers based on their skills and bio.
- **Real Profiles**: Showcase your tech stack (React, Node, Python, you name it) and your developer journey.
- **Cloud Connections**: Manage your matches and pending requests in real-time.
- **Glassmorphism UI**: A sleek, dark-mode design with smooth animations that feels premium.
- **Mobile First**: Fully responsive because we're often coding (or swiping) on the go.

## 🚀 Tech I used
- **Frontend**: React 19 (the latest!), Redux Toolkit for clean state management, and React Router for fast navigation.
- **Styling**: Tailwind CSS + DaisyUI for that beautiful, modern look without the CSS headache.
- **Backend**: Node.js & Express (located in a separate repo).
- **Database**: MongoDB/Mongoose.

## 🌐 Deployment (Vercel)

1. **Environment Variables**: Add `VITE_API_BASE_URL` pointing to your deployed backend URL.
2. **Build Command**: `vite build`
3. **Output Directory**: `dist`
4. **Deploy**: Push to GitHub and import to Vercel.
